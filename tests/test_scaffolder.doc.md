# `tests/test_scaffolder.py`

## What it does
Unit tests for the scaffolder: `slugify`, `package_name`, `ScaffoldSpec`
validation, and the `Scaffolder` end-to-end (using `tmp_path`).

## Dependencies
- `pytest`
- `sin_mcp_server_builder.scaffolder`

## Usage
```bash
pytest tests/test_scaffolder.py -q
```
